== reAuctions

A user-friendly e-sourcing application, thought to optimize procurement processes. It provides RFPs management module and also a live auction to find out which Supplier offers the best bid.

* Ruby version: ruby 2.2.3p173 (2015-08-18 revision 51636)

* Rails version: Rails 4.2.4

* Stack: 
    * Ruby on Rails
    * Javascript
    * D3.js
    * jQuery
	

<tt>rake doc:app</tt>.
